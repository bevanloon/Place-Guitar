Background

Inspired by a tweet, I hacked this together over a few evenings.

It is written in Python, and hosted on Google App Engine.

Because the App Engine image service doesn't really allow full control over images, you will probably get an image that has been slightly cropped one way or another. App Engine will resize images, but it always respects the aspect ratio. That doesn't allow for true scalling of images, so I've faked it by sizing the image up and then cropping it to the correct dimensions. In retrospect, though it annoyed me while I was trying to hack this together quickly, it is probably a good thing because it would look pretty strange if I started returning images that were scaled without regard to the aspect ratio - sort of reminiscent of those badly designed web pages that we've all come across where an 800 x 600 image was forced into an img tag with 400 x 400 dimensions.

The source, in all its glory (yes that is irony), is on github. I only really play in Python part time and I expect the code reflects that.
